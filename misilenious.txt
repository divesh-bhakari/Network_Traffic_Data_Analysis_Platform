how to run this project 
------------------------
1) python --version (it should be above 3.8)
2) unzip and go to project folder - cd NTDAP
3) pip install -r requirenments.txt
4) cd backend
5) python main.py - to run the backend on localhost http://127.0.0.1:5000/
6) open the index.html from frontend using file explorer
7) uoload a pcap file


how does it works
------------------

Step 1 — parser.py
Job: Read the PCAP file and extract packet data
*********************************************** 

Takes the uploaded .pcap file
Uses Scapy to open and read every packet
Loops through each packet one by one
For each packet it extracts fields like:

Source IP, Destination IP
Protocol (TCP, UDP, ICMP, DNS etc)
Ports, TTL, flags, packet size

Puts all of this into a Pandas DataFrame (like a spreadsheet)
Output: a table where each row = one packet, each column = one feature



Step 2 — cleaner.py
Job: Clean up the data
**********************
Takes the DataFrame from parser.py
Removes duplicate packets
Fills in missing values (empty cells get 0 or "Unknown")
Removes broken packets with zero length
Makes sure numbers are actually stored as numbers
Output: a cleaner, more reliable DataFrame


Step 3 — eda.py
Job: Calculate statistics
*************************
Takes the cleaned DataFrame
Counts things and calculates averages:

Total packets, total bytes
Average/min/max packet size
Which protocol was used most
Which IP sent the most packets
Which ports were used most


Output: a dictionary of statistics (just numbers and labels)


Step 4 — analyzer.py
Job: Find unusual packets using Machine Learning
***************************************************
Takes the cleaned DataFrame
Picks numeric columns: packet size, payload size, TTL
Feeds them into Isolation Forest (scikit-learn)
The model learns what "normal" packets look like
Flags packets that don't fit the pattern as anomalies
Labels each packet: 1 = normal, -1 = anomaly
Output: anomaly count, percentage, and a label for every packet


Step 5 — visualizer.py
Job: Draw charts and save them as images
****************************************
Takes the cleaned DataFrame and anomaly results
Creates 6 charts using Matplotlib:

Protocol distribution (bar chart)
Top source IPs (horizontal bar)
Packet size distribution (histogram)
Traffic over time (line chart)
Top destination ports (bar chart)
Normal vs anomaly packets (scatter plot)


Saves each chart as a .png file in the results/ folder
Output: file paths to the 6 saved chart images


Step 6 — interpreter.py
Job: Write a plain-English explanation
***************************************
Takes the statistics and anomaly results
Uses if/else logic to build sentences:

"The capture contains X packets..."
"Most traffic is TCP..."
"X anomalies detected, rate is low/medium/high..."


Assigns a severity level: NORMAL, MEDIUM, or HIGH
Output: a dictionary of text paragraphs


Step 7 — main.py
Job: Connect everything together as a web server
************************************************
Runs a Flask web server on port 5000
Waits for the frontend to send a PCAP file
When a file arrives it calls each module in order:
